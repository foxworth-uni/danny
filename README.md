# Danny

Smart bundler analyzer for modern web applications.

## Installation

### Homebrew (macOS)

```bash
brew install foxworth-uni/danny
```

### From Source

```bash
cargo install --path crates/danny-cli
```

## Usage

```bash
danny --help
```

## Development

### Building

```bash
cargo build --release
```

### Testing

```bash
cargo test
```

### Running Locally

```bash
cargo run -p danny-cli -- [ARGS]
```
